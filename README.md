## Installation

``` bash
$ npm install @sukka/hexo-util-rs
// or
$ yarn add @sukka/hexo-util-rs
// or
$ pnpm add @sukka/hexo-util-rs
```

## Usage

``` js
const util = require('@sukka/hexo-util-rs');
```
